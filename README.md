# Kzares Portfolio

Retro movie-theater inspired developer portfolio for Jorge Casares, built with Astro, React, Tailwind CSS, and content collections.

## Features

- Astro static site generation
- React interactive countdown and language switcher
- English and Portuguese content
- Dynamic initial language selection for Brazil/Portuguese visitors
- Markdown-driven projects collection
- Retro theater visuals with SVG, film grain, cassette/project screening UI, and no CSS background gradients
- SEO metadata, Open Graph, Twitter cards, canonical URL, hreflang, and JSON-LD

## Stack

- Astro
- React
- Tailwind CSS
- TypeScript content config
- GitHub Pages

## Commands

```sh
npm install
npm run dev
npm run build
npm run preview
```

## Deployment

The site is configured for GitHub Pages at:

https://kzares-dev.github.io/

Deployments run through GitHub Actions using `.github/workflows/deploy.yml`.
